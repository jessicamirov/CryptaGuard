About Us

Our solution involves the development of an accessible website for encrypting files using an asymmetric method, ensuring secure and confidential file transfer.

We utilize peer-to-peer communication, enabling users to transfer files securely without reliance on central entities. Our website serves private users, businesses, and organizations, facilitating secure file transfer for sensitive documents.

How It Works

  1. Connect to the secure peer-to-peer network.
  2. Generate encryption keys securely upon connection.
  3. Encrypt files and securely send them to connected peers.
  4. Receive encrypted files from peers for secure storage or transmission.
